# 📖 Mi Biblioteca Personal

¡Bienvenido a **Mi Biblioteca Personal**! Esta es una aplicación web minimalista y elegante diseñada para los amantes de la lectura que quieren gestionar su colección de libros digital de forma sencilla, visual y con un estilo moderno.

## ✨ Características principales

* **🔍 Búsqueda Inteligente:** Encuentra libros por título o autor con sugerencias en tiempo real (Autocompletado).
* **📚 Gestión por Estados:** Organiza tus libros en tres categorías claras:
    * *Leyendo ahora*: Para los que tienes entre manos.
    * *Ya leídos*: Tu historial de conquistas literarias.
    * *Pendientes*: Tu lista de deseos y próximas aventuras.
* **🎨 Diseño Neomórfico:** Interfaz basada en el estilo *Neumorphism*, con sombras suaves y relieves que dan una sensación táctil y moderna (inspirado en apps de productividad de alto nivel).
* **🌓 Modo Oscuro/Claro:** Cambia el tema con un solo clic para proteger tu vista durante las lecturas nocturnas.
* **📱 Diseño Responsivo:** Adaptado para que puedas consultar tu biblioteca tanto en el ordenador como en tu móvil.
* **🖼️ Detalles del Libro:** Haz clic en cualquier portada para ver la descripción completa y una imagen en alta resolución.

## 🛠️ Tecnologías utilizadas

Este proyecto ha sido desarrollado utilizando tecnologías web estándar para garantizar ligereza y velocidad:

* **HTML5:** Estructura semántica.
* **CSS3 (Custom Properties & Flexbox/Grid):** Diseño neomórfico avanzado y transiciones suaves.
* **JavaScript (Vanilla JS):** Lógica de la aplicación, gestión de la base de datos local y llamadas a la API de libros.
* **Google Books API:** Para obtener información precisa de millones de títulos de forma automática.

## 🔒 Privacidad y Almacenamiento

Esta aplicación ha sido diseñada bajo el concepto **Offline-First**. 

* **Sin Cuentas:** No necesitas registrarte ni compartir tus datos personales para usar la biblioteca.
* **Almacenamiento Local (LocalStorage):** Toda tu información (libros, notas, puntuaciones y preferencias de tema) se guarda exclusivamente en tu navegador. Esto significa que tus datos nunca salen de tu dispositivo, garantizando una privacidad total.
* **Portabilidad:** Si deseas llevar tus datos a otro lugar, puedes usar la función de **Exportar a PDF** para generar un informe completo de tu colección en segundos.

---
*Nota: Al usar LocalStorage, si borras los datos de navegación o el historial de tu navegador, la biblioteca se reiniciará. ¡Asegúrate de exportar tu PDF si vas a hacer una limpieza profunda de tu equipo!*
